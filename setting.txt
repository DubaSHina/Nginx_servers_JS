Перед запуском требуется : 

- Установить node.js
______________________________

- Запуск команд *.js - node "Имя сервера".js
- Остановить выполнение команды *.js - нажимаем клавиши "Ctrl" + "C"

Чтобы всё запустить требуется:

1.1 Перейти в директорию /server1 - запустить server1.js 
1.2 Перейти в директорию /server2 - запустить server2.js
1.3 Перейти в директорию /server3 - запустить server3.js

2.1 Если хотим посмотреть , как это работает в терминале ( Ubuntu )  , то выполняем команду :
while sleep 1; do curl http://localhost:8888; done
2.2 Можно также перейти в браузер и вбить строку : 
http://localhost:8888/
